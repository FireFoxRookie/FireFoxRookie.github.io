# 编写说明

安装注意事项

建议全局安装vuepress，同时使用yarn进行初始化

# 项目初始化

TIP

确保全局安装了VuePress

第一次使用时需要在项目的根目录下，也就是'/前端笔记'目录下运行如下命令:



yarn install 或 npm install
1

TIP

推荐使用yarn进行安装，npm安装时会有的问题

# markdown编写目录

笔记的内容使用markdown进行编写，在/docs/library目录下新建自己的markdown文件

# 页面注册

编写的markdown文档需要进行路由注册才能显示，注册方法：在/docs/.vuepress/config.js中进行如下编码:




 














module.exports = {
  ...
  themeConfig: {
    ...
    sidebar: {
      '/library/': [
         {
           title: '知识点记录',
           collapsable: false,
           children: [
             '自己创建的markdown文件名(不加后缀.md)'
           ]
         }
      ]
    }
  }
}
